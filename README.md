# TODOAPP

Ứng dụng **TODOAPP** được phát triển trong khuôn khổ môn học **Phát triển ứng dụng trên thiết bị di động - NT118**.
Dự án được xây dựng bằng ** Java trên Android Studio**, với mục tiêu giúp người dùng **tạo, quản lý và theo dõi công việc hằng ngày** một cách dễ dàng và tiện lợi.

## Công nghệ sử dụng
- **Ngôn ngữ:** Java
- **IDE:** Android Studio

## Thông tin
- Môn học: **Phát triển ứng dụng trên thiết bị di dộng - NT118**
- Dự án: **TODOAPP**
- Nền tảng: **Android**
