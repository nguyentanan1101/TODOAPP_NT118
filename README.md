# TODOAPP 📝

Ứng dụng **TODOAPP** được phát triển trong khuôn khổ môn học **Phát triển ứng dụng trên thiết bị di động (NT118)**.  
Dự án được xây dựng bằng **Java trên Android Studio**, với mục tiêu giúp người dùng **tạo, quản lý và theo dõi công việc hàng ngày** một cách dễ dàng và tiện lợi.

## 💡 Chức năng chính
- Thêm, sửa, xóa công việc  
- Đánh dấu công việc đã hoàn thành  
- Giao diện thân thiện, dễ sử dụng  

## 🛠️ Công nghệ sử dụng
- **Ngôn ngữ:** Java  
- **IDE:** Android Studio  
- **Cơ sở dữ liệu:** SQLite (hoặc Room nếu có)
